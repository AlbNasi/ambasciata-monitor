Piccolo esperimento di vibecoding con Claude (niente Claude Code per ora).

L'obiettivo è una piccola app per mac che sia in grado di raccogliere tutte le informazioni pubblicate dalle varie ambasciate italiane e di riportarle su un'unica UI con una mappa per selezionare i paesi che interessano e una bacheca con i vari comunicati. Il progetto sarà quindi composto da uno scraper che colleziona i dati online, un database che colleziona le informazioni e infine una interfaccia grafica, tre elementi abbastanza diversi per testare le capacità AI.

Vedremo in futuro se ha senso estenderlo in qualche modo.
